# Yshi-Mincho简介
根据思源宋体修改，利用思源宋体映射字库，选择传承字形创作
# 授权
采用SIL Open Font License 1.1授权。
# 注意
本字体难免有字形错误，且部分字形不统一，本人将逐个版本进行修正
# 特色
支持旧字形写法，如為和爲這種字，默認選擇爲，真和眞這種，默認選擇眞，但真的本字為與眞區分，因此部分写法采用异体写法
